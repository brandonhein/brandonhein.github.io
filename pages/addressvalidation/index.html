Address Validation Service API

Similar to the Location Lookup API I made, I leveraged AWS Lambda and the AWS API Gateway to create a Serverless Application, that aggregates/orchestrates calls to my Location Lookup API, USPS Address API, Google Address API, and US Census tract API. It compiles a response used for any application that needs to know/validate addresses
